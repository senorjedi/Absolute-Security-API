# Absolute-Security-API
Custom API call to unenroll and unfreeze the device with PIN in PowerShell with PowerShell scripts to force call the device to the console.  

When unenrolling a device to ensure that persistence is deactivated. Download the ABTPS.exe from the console and run the Force call script in admin to force call to the console to ensure that all pending actions have gone through and that persistence is deactivated.
